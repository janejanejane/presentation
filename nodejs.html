Ryan Dahl published Node.js in 2009 at JSConf EU to propose a different approach in I/O processing

Traditional I/O Approach

Programs were taught to be linear

Program:


printf("What's your name? \n");
scanf("%s", &str);
printf("Hello %s!\n", str);

Output:


What's your name?
Jean
Hello Jean!

Node.js I/O Approach

Programs should not halt execution

Program:


setTimeout(function(){
	console.log("What's your name? ");
}, 3000);
console.log("Hello ");

Output:


Hello
[3 seconds later]
What's your name?

Getting Started

Simple server 


// file is server.js
var http = require("http");

http.createServer(function(request, response) {
  response.writeHead(200, {"Content-Type": "text/plain"});
  response.write("Hello World");
  response.end();
}).listen(8888);

Run it:


node server.js
